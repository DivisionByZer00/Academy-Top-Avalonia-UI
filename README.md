# Academy-Top-Avalonia-UI

#### Решения домашних заданий "Разработка кроссплатформенных GUI-приложений с Avalonia UI"

## Homework 01

### Тема: "Анатомия GUI приложения. Введение в Avalonia UI"

**Задание**: Создайте новый проект Avalonia с названием **Homework1_AboutMe**. Приложение должно содержать одно окно со следующими элементами:

**Заголовок окна:** «Портфолио: [Ваша Фамилия]».

**Размер окна:** Окно должно быть компактным (примерно 300x450) и фиксированным (запретить растягивание CanResize=»False»).

**Содержимое окна** (внутри StackPanel):
- *Заголовок:* Ваше имя (Крупный шрифт, выравнивание по центру).
- *Описание:* Краткий текст о себе (например: «Студент 3 курса, люблю C# и котиков»). Текст должен переноситься на новую строку, если он длинный (используйте свойство TextWrapping=»Wrap»).
- *Секретная информация:* TextBlock, который изначально скрыт (свойство IsVisible=»False»).
- *Панель кнопок*: Две кнопки, расположенные внизу.

**Логика кнопок**

**1. Кнопка «Показать секрет»:**      
   а. При нажатии скрытый текст становится видимым;      
   б. В тексте может быть интересный факт о вас;      
   в. *Подсказка:* Чтобы обратиться к тексту из C#, не забудьте дать ему имя (x:Name). В коде используйте свойство .IsVisible = true;.      
**2. Кнопка «Выход»:**      
   а. При нажатии приложение должно закрываться:      
   б. *Подсказка:* Для закрытия текущего окна используйте команду this.Close(); в C# коде.

## Homework 02

### Тема: "Интерактив и ввод данных. Элементы управления"

**Задание**: Разработайте приложение «Форма заказа пиццы».  
Представьте, что вы пишете интерфейс для терминала самообслуживания в пиццерии. Пользователь должен ввести свои данные, выбрать пиццу, настроить количество и подтвердить заказ.

**Требования к окну:**
- *Заголовок:* «PizzaOrder: [Ваша Фамилия]»
- *Размеры:* Фиксированные (например, 400x600), CanResize=»False».
- *Фон:* Светлый (например, White или Beige).

**Содержимое окна:**
- Блок «Клиент»;
- Блок «Выбор пиццы» (Один из вариантов);
- Блок «Добавки» (Множественный выбор);
- Блок «Количество»;
- Блок «Итог».

**Подсказки:**
- Для проверки пустоты строки используйте: string.IsNullOrWhiteSpace(MyTextBox.Text)
- Помните, что NumericUpDown.Value и CheckBox.IsChecked могут быть null, поэтому используйте безопасное приведение или проверку (например, CheckCheese.IsChecked == true).
- Чтобы сделать отступы между блоками, используйте свойство Margin у элементов или оберните группы элементов в отдельные StackPanel.

## Homework 03

### Тема: "Мастер компоновки"

**Задание**: Разработайте приложение «Карточка сотрудника».      
Вам нужно сверстать окно, которое выглядит как анкета сотрудника в корпоративной системе. Главное условие — использовать контейнер Grid для управления макетом.

**Требования к окну:**
- *Заголовок:* «Сотрудник: [Ваша Фамилия]»
- *Размеры:* Фиксированные (например, 500x350), CanResize=»False».
- *Фон:* Светло-серый (например, WhiteSmoke или #F0F0F0).

**Структура окна:**
Корневой элемент должен быть Grid. Вам нужно «нарезать» его следующим образом:
- Столбцы (Columns);
- Строки (Rows).

**Содержимое окна:**
- Фото сотрудника (Левый столбец);
- Имя сотрудника (Правый столбец, Верхняя строка);
- Данные анкеты (Правый столбец, Средняя строка);
- Кнопки управления (Нижняя строка).
